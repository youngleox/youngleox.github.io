title: "Augmented reality powers a cognitive assistant for the blind"
collection: publications
permalink: /publication/ar
date: 2018-11-27
venue: 'ELife'

---
We developed an intuitive user interface with wearable augmented reality devices for the blind: each object in the environment has a voice and communicates with the user on command. With minimal training, this system enables blind users to traverse an unfamiliar multi-story building on their first attempt.
